Consideraciones¶

  • Esto quizás sea para usuarios algo avanzados, especialmente porque no habrá mucha explicación.

  • La idea de esto, es que puedas tener una única copia del juego para ser usada en diferentes proyectos sin problemas.

    • Nada de tener que copiar el juego en otro lado y ahí instalar un proyecto.

Requisitos¶

  • Lee antes de hacer, así no cometerás errores.

    • Y pensar, que no es delito. @,@

  • Tener instalado Link Shell Extension.

    • Obviamente puedes utilizar otros programas que hagan lo mismo o realizar todo en el Símbolo del sistema usando MKLINK.

  • Tener activadas las opciones para ver las Extensiones de archivos + Archivos y carpetas ocultos para que todo pueda ser un poco más fácil.

  • Tener una copia limpia del juego (siempre se tratará de tomar como referencia una copia original, completa y actualizada).

    • Preferiblemente en español y que nunca se haya iniciado.

  • Tener una idea de como instalar el proyecto que quieras probar.

  • Tener una idea de los archivos que se editarán/reemplazarán.

Proceso de Instalación¶

  1. Ve a donde tienes el juego base (o donde quieras realmente) y crea una carpeta llamada Proyecto X.

  2. Dentro de la carpeta Proyecto X, tendrás que instalar el Proyecto X.

  3. En la carpeta del juego base, seleccionas todo excepto:

  • ejemplo.dll

  • ejemplo_carpeta

  1. Haces click derecho y seleccionas Seleccionar el origen del Vínculo.

  2. Dentro de la carpeta Proyecto X, haces click derecho, seleccionas Soltar como… y finalmente Vínculo Simbólico.

  3. Ahora crea una carpeta ejemplo_carpeta.

  4. Dentro de la carpeta ejemplo_carpeta del juego base, seleccionas solamente:

  • ejemplo.cfg

  1. Haces click derecho y seleccionas Seleccionar el origen del Vínculo.

  2. Dentro de la carpeta ejemplo_carpeta de Proyecto X, haces click derecho, seleccionas Soltar como… y finalmente Vínculo Simbólico.

  3. Si todo está bien, ahora puedes iniciar Proyecto X y que este funcione sin problemas.

  • Y luego de cerrar Proyecto X, puedes iniciar tu juego original sin problemas. :P

Nota

  • El Proceso de Instalación es genérico, no en todos los casos será así de simple. e,é

  • Si tengo una carpeta Ejemplo-A donde dentro necesito todo como Enlace Simbólico, pero justo en esta carpeta se generan archivos cuando inicio el Juego / Proyecto, debo crear esta carpeta y dentro poner individualmente los archivos originales como enlaces simbólicos.

    • Todo para que entre Juego Original y Proyecto X haya una separación al 100%
